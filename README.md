# AgrauData — Portfolio Website

Portfolio profesional de AgrauData, Automation & Data Intelligence Consultant.

## Stack

- **Framework**: Next.js 14 (App Router)
- **Estilos**: Tailwind CSS
- **Tipado**: TypeScript
- **Animaciones**: Framer Motion
- **Despliegue**: Vercel

## Desarrollo local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000)

## Deploy

Conectado a Vercel — cada push a `main` despliega automáticamente.
